--------------------------------------------------------------------------------
                                Introduction
--------------------------------------------------------------------------------
Profiler is a set of helper functions that aid in creation of Install Profiles. 
It allows you to quickly and easily create new install profiles, as well as have
'Sub' Profiler Install profiles, where one Profiler Install profile inherits 
from and extends another Profiler Install profile.

--------------------------------------------------------------------------------
                                Installation
--------------------------------------------------------------------------------
Profiler depends on Install Profile API [1], so be sure you download that first.
Place Profiler inside your profiles directory within the Drupal distribution.
Look at the Profiler Example profile [2] to see how you create your own Install
profile.

[1] http://drupal.org/project/install_profile_api
[2] http://drupal.org/project/profiler_example
